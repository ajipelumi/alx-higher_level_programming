The ``1-my_list`` module
======================

Using ``MyList``
-------------------

First import ``MyList`` from the ``1-my_list`` module:
    >>> MyList = __import__('1-my_list').MyList

Now use it:

    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

    >>> print(my_list)
    [1, 4, 2, 3, 5]

    >>> print(my_list.index(5))
    4

    >>> my_list.append(2)
    >>> print(my_list.count(2))
    2

    >>> my_list.pop(1)
    4

    >>> print(my_list)
    [1, 2, 3, 5, 2]
